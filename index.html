Getting Started

Crobrew is still in its early stages, so some features might not work as expected. It is designed to be a simple and lightweight package manager for multiple platforms.

Installation

To install Crobrew, use the following command:

go install github.com/chersbobers/crobrew@latest

Ensure Go is added to your PATH:

echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bashrc && source ~/.bashrc

Platform-Specific Instructions

ChromeOS

  1. Enable Linux (Beta) in ChromeOS Settings.
  2. Install Go:
    3. sudo apt-get update && sudo apt-get install golang-go

Linux

Install Go using your distribution's package manager:

  • Ubuntu/Debian: sudo apt-get install golang-go
  • Fedora: sudo dnf install golang
  • Arch Linux: sudo pacman -S go

macOS

Install Go using Homebrew:

brew install go

Windows

Download Go from golang.org and run the installer. Add Go to your PATH:

setx PATH "%PATH%;%USERPROFILE%\go\bin"

Usage

Crobrew can be used in two modes: command-line and interactive.

Command-Line Mode

# Update package list
cro update

# Search for packages
cro search firefox

# Install a package
cro install firefox

# Remove a package
cro remove firefox

Interactive Mode

Run cro without arguments to start interactive mode, which provides a menu-based interface.

Features

  • Automatic package manager detection for all major platforms.
  • Update package lists.
  • Search for packages.
  • Install and remove packages.
  • Supports multiple package managers (apt, dnf, pacman, brew, chocolatey, snap, WSL).

Supported Package Managers

OS Package Managers
ChromeOS apt, snap
Linux apt, dnf, pacman, snap
macOS brew
